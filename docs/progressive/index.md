这份渐进式指南会从基础的 TypeScript 语法开始，从零开始带你编写一个拥有完整交互功能的私人助手 Bot。

其实这也是一份很适合用来入坑 TypeScript 的材料 XD

虽然名义上是从零开始的指南，但是对于 TypeScript 本身，并不会一次性着墨太多，很多关于 TypeScript 本身的细节都会藏在后面的实践流程里，所以我还是更建议有基础以上的编程能力的开发者去尝试以这篇指南作为 TypeScript 的入门，当然如果你想挑战一下自己，也不是不可以。

那么当然就是... START YOUR ENGINE！
