Sakiko 的社区项目主要分为三类：

- 适配器
- 插件
- 其他

其中，**适配器** 主要负责打通 Sakiko 和各个平台的连接，**插件** 主要是与框架本身强相关的功能扩展，**其他** 则是和框架本身不强相关但是功能上可以配合或互补的项目。

目录以项目**去掉前缀**后的名称的拼音首字母由 A-Z 排列，同一首字母下按照列入的顺序自上而下排列。

社区项目可以使用如下的命名格式：

- sakiko-adapter-\*（适配器）
- sakiko-plugin-\*（插件）

社区项目可以自行在 Github 等仓库页面上编写文档，也可以直接将文档推送至当前的社区内容文档分类中。
